# CP-ABE-test
CP-ABE测试加解密操作和密钥生成操作的性能<br>
一、测试环境配置<br>
1、在ubuntu中安装第三方库libbswabe-0.9和libbswabe-proxy，使用以下命令：<br>
cd libbswabe-0.9<br>
./configure<br>
make<br>
make install<br>
cd libbswabe-proxy<br>
./configure<br>
make<br>
make install<br>
